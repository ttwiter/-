# Proyek ttwiter/-

## Deskripsi
Proyek ini dibuat menggunakan JavaScript, CSS, dan HTML.

## Cara Menjalankan
1. Pastikan Node.js sudah terinstall.
2. Jalankan perintah berikut di terminal:
   ```sh
   node server.js
   ```
3. Buka browser dan akses `http://localhost:3000`.
